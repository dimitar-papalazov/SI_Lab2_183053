### Тест случаи според критериумот Multiple condition

* (deg >= 0 && deg < 360)
  * T && T
  * F && T
  * T && F
  * F && F same for both


* (min < 0 || min > 59)
  * T || T not possible
  * T || F
  * F || T
  * F || F same as the upper 2 cases, will use the 2nd
  
  
* (sec < 0 || sec > 59)
  * T || T not possible
  * T || F
  * F || T
  * F || F same as the upper 2 cases, will use the 2nd
  
  
* (deg == 360)
  * T
  * F
  
  
* (min == 0 && sec == 0)
  * T && T
  * T && F
  * F && T
  * F && F same as the upper 2 cases, will use the 2nd
